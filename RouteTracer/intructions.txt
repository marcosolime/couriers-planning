Route Tracer

Description
- this is a python script that let you visualize the routes of the couriers on a map
- it tries to recover the xy location of the clients given the distance matrix
- note that it does not recover the absolute position of the clients, rather respects the relative distances between nodes

How to use the program
- put the instance file (.dat) under the 'res' folder
- put the result file (.json) under 'inst' folder
- run the program to get the map as a png image under 'out' folder

How to run the program
- go under 'RouteTrace'
- type: python ./src/main.py ./inst/<instXY.dat> ./res/<XY.json>
